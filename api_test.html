Create Task

This form submits directly to your FastAPI endpoint at http://127.0.0.1:8001/api/tasks.

Before using this form:

  1. Open a terminal in your project folder.
  2. Activate venv: .\env\Scripts\Activate.ps1 (you should see (env)).
  3. Start FastAPI: uvicorn fastapi_app.main:app --reload --port 8001
Allowed values: low | medium | high
Format: YYYY-MM-DD
The response JSON will open in a new tab.

Tip: You can also test quickly from the address bar: open http://127.0.0.1:8001/api/tasks to list tasks.
